install in Hermes / Codex
  1. download & unzip the pack into your pets dir: 
    unzip tiny-dino-<id>.zip -d ~/.codex/pets/
  2. activate it: run petdex select, or in Codex/Hermes go Settings → Appearance → Pets → Select, then /pet to wake it.
  3. want it in the public gallery? petdex submit ~/.codex/pets/tiny-dino-<id>
